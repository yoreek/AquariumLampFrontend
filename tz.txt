VueJS + vite + vuetify + TypeScript + fetch API web application for manage aquarium lamp.

Лампа собрана на ESP32 и управление будет по API.
Лампа при включении может пытаться подключиться к точке доступа, если не получилось или не настроен доступ то сама создает точку доступа с названием AquariumLamp, IP 192.168.4.1 и без пароля.
Язык один английский.

У лампы есть 5 каналов, яркость каждого можно регулировать 0-100% .
Режим работы будет 3 варианта:
1. По расписанию
2. Ручной
3. Выключено

Когда по расписанию то есть 10 элементов в расписании, каждый имеет настройки:
1. Enabled/disabled
2. Время ( часы/минуты суток)
3. Яркость 0-100%

Такими настройками задаются основные точки времени в которых указанная яркоcть если enabled.
Промежуточное значение яркости между временными точками вычисляется апроксимацией.


Будет главная страница:
1. верхняя панель содержит текущую дату и время, справа иконка для переходя в настройки
2. ниже главная панель на которой будет график 5 цветных линий для каждого цвета, колторый отображает якрость в течении суток согласно настроек расписания, только enabled, апроксимаци и сделать плавные переходы, на графике отображается еще верикальная линии согласно текущего времени
3. ниже будет панель с ручной установкой яркости, будет 5 слайдеров с настройкой каждого канала и там же будет переключатель режимов (3 положения)
4. ниже будут настройки расписания, там 10 элементов, каждый элемент состоит
4.1. свитч enabled/disabled
4.2. время (часы/минуты) в 12 часовом или 24 часовом формате (режим будет в настройках)
4.3. яркость 0-100% слайдер
5. Страницы настройками как на второй картинке примерно.
5.1. Wifi конфигурация
5.1.1. Режим работы точка доступа или клиент или авто (пытается в режиме клиента подключится к точке доступа, если не получилось то переключается в режим точки доступа)
5.1.2. Отдельный блок с настройкми точки доступа есть настройки SSID,  IP и пароль.
5.1.3. Отдельный блок с настройками клиента - селектор для выбора точки доступа динамический (SSID, Signal Level), кнопка Scan для принудительного обновления списка, пароль.
5.2. Настройки времени и синхронизации
5.2.1. Поле дата и поле время для настройки времени, если не работает синхронизация по NTP
5.2.2. Свичт Auto Time Sync (NTP) enabled/disabled
5.2.3. Поле NTP server
5.2.4. Поле Timezone - селектор с выбором таймзоны, по API получаем список в JSON id/name
5.2.5. Настройка формата времени 12/24
5.3. панель с информацией о устройстве и там кнопка перегрузить.
